# SMD Quiz 8 Android Application

This project is developed for Quiz 8 of Mobile Software Development.

## Features Implemented

### 1. Local Notification
- Navigate to Local Notification Activity
- Send Local Notification
- Notification contains:
  - Title
  - Message
  - Notification Icon

---

### 2. Firebase Notification & Analytics
- Subscribe user to Firebase Topic
- Firebase Analytics event logging implemented

---

### 3. Camera Functionality
- Open mobile camera
- Capture photo
- Display captured image on screen
- Runtime permission handling added

---

### 4. Media Activity
#### Audio Button
- Plays audio/song using MediaPlayer

#### Media Button
- Opens YouTube video using Intent

---

### 5. Google Maps Functionality
- Google Maps API integrated
- Map displayed successfully
- User can select location on map
- Selected location information displayed

---
